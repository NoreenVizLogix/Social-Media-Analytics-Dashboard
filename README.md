# 📌 Project Overview

The **Social Media Analytics Dashboard** is an interactive Power BI report that enables users to analyze social media performance across multiple platforms including Instagram, Facebook, LinkedIn, TikTok, YouTube, and X (Twitter).

The dashboard transforms raw engagement data into meaningful business insights through interactive KPIs, trend analysis, geographical reporting, and performance comparisons.

It is designed for:

- 📊 Business Analysts
- 📈 Marketing Teams
- 📱 Social Media Managers
- 📢 Digital Marketing Agencies
- 💼 Business Decision Makers

---

# 🛠 Tech Stack

The dashboard was built using the following technologies:

- 📊 **Power BI Desktop** – Dashboard development
- 🔄 **Power Query** – Data cleaning and transformation
- 🧠 **DAX (Data Analysis Expressions)** – KPIs and calculated measures
- 🔗 **Data Modeling** – Relationship management
- 📈 **Data Visualization** – Interactive charts, maps, slicers, and heatmaps
- 📁 **File Format** – `.pbix`

---

# 📂 Data Source

The dashboard is based on a social media performance dataset containing engagement metrics collected from multiple platforms.

### Dataset Includes

| Field | Description |
|--------|-------------|
| Platform | Social media platform |
| Country | Audience location |
| Date | Posting date |
| Content Type | Video, Image, Live, Article, PDF |
| Views | Number of views |
| Likes | Total likes |
| Comments | Total comments |
| Shares | Total shares |
| Engagement | Overall engagement |
| Engagement Rate | Engagement percentage |

# 📌 Business Problem

Organizations publish content across multiple social media platforms every day, generating large volumes of engagement data.

Without a centralized analytics solution, it becomes difficult to answer important business questions such as:

- Which platform generates the highest engagement?
- Which content type performs the best?
- Which countries have the most active audience?
- What is the best time to publish content?
- How has engagement changed over time?

Manual analysis is time-consuming and often lacks actionable insights.

---

# 🎯 Dashboard Goal

The objective of this dashboard is to provide a centralized reporting solution that helps users:

- Monitor overall social media performance
- Compare multiple platforms
- Analyze audience engagement
- Identify high-performing content
- Track performance trends
- Support data-driven marketing decisions
